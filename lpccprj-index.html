LiDAR Point Cloud


Fig. 1: Applications of Static Point Cloud (SPC), Dynamic Point Cloud (DPC), and dynamically-acquired LiDAR Point Cloud (LPC). Static point clouds (SPC) and dynamic point clouds (DPC) are primarily employed in applications related to human perception, such as 3D object display in arts and culture and real-time 3D telepresence . On the other hand, dynamically-acquired LiDAR point cloud (LPC) is mainly used for machine perception applications, specifically in robotics and autonomous driving.
